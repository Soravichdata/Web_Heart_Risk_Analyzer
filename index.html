Dataset Features
รายละเอียดตัวแปร
คำอธิบายตัวแปร 14 ตัวแปรในชุดข้อมูล Cleveland Heart Disease Dataset
ตัวแปร ชื่อเต็ม ประเภท ช่วงค่า คำอธิบาย
age อายุ ตัวเลข 29–77 อายุของผู้ป่วย (ปี)
sex เพศ หมวดหมู่ 0, 1 1 = ชาย, 0 = หญิง
cp Chest Pain Type หมวดหมู่ 0–3 ประเภทอาการเจ็บหน้าอก (0=typical, 3=asymptomatic)
trestbps Resting Blood Pressure ตัวเลข 94–200 ความดันโลหิตขณะพัก (mmHg)
chol Cholesterol ตัวเลข 126–564ระดับคอเลสเตอรอลในเลือด (mg/dl)
fbs Fasting Blood Sugar หมวดหมู่ 0, 1 1 = น้ำตาลในเลือด >120 mg/dl
restecg Resting ECG หมวดหมู่ 0–2 ผลตรวจ ECG ขณะพัก
thalach Max Heart Rate ตัวเลข 71–202 อัตราหัวใจสูงสุดที่ทำได้
exang Exercise Angina หมวดหมู่ 0, 1 1 = เจ็บหน้าอกเมื่อออกกำลังกาย
oldpeak ST Depression ตัวเลข 0–6.2 การลดลงของ ST segment ขณะออกกำลังกาย
slope ST Slope หมวดหมู่ 0–2 ความชันของ ST ขณะออกกำลังกาย (0=downsloping)
ca Major Vessels ตัวเลข 0–4 จำนวนหลอดเลือดหลักที่เห็นจาก fluoroscopy
thal Thalassemia หมวดหมู่ 0–3 ผลการตรวจ thalassemia
target ผลการวินิจฉัย หมวดหมู่ 0, 1 1 = มีความเสี่ยงโรคหัวใจ, 0 = ไม่มีความเสี่ยง
